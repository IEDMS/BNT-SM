gd_destroyproxy

Destroys the local copy of the user's Globus proxy certificate.

 

Syntax

gd_destroyproxy

 

Description

This command deletes the local copy of the Globus proxy certificate for the user's credentials at the location specified by the cog.properties file.

 

See also

gd_createproxy, gd_proxyinfo, gd_certinfo
