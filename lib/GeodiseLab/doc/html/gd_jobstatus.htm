gd_jobstatus

Gets the status of a Globus GRAM job.

 

Syntax

status = gd_jobstatus(jobhandle)

 

Description

status = gd_jobstatus(jobhandle) returns the status of a Globus GRAM job, where status:

                        -1 is UNKNOWN

             1 is PENDING

                         2 is ACTIVE

                         3 is DONE

             4 is FAILED

                         5 is SUSPENDED

                         6 is UNSUBMITTED

 

Notes

A valid proxy certificate is required to query a GRAM job.

 

See also

gd_createproxy, gd_jobsubmit, gd_jobkill
