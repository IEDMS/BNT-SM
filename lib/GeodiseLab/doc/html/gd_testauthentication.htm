gd_testauthentication

Tests authentication with a Globus resource.

   

Syntax

success = gd_testauthentication(host)

 

Description

success = gd_testauthentication(host) where success is the outcome of authentication with the Globus server host. The value of success is 1 on success and 0 on failure.

   

Example

success = gd_testauthentication('server.domain.com');

 

Notes

A valid proxy certificate is required to use this function.
